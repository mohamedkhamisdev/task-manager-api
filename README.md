# Task Manager API

A production-ready ASP.NET Core Web API for managing tasks with full CRUD functionality.

## About

TaskManagerAPI is a RESTful web service built with ASP.NET Core 8 and Entity Framework Core. It demonstrates fundamental REST API principles with a complete task management system.

## Features

- ✅ Complete CRUD operations (Create, Read, Update, Delete)
- ✅ RESTful API design with standard HTTP methods
- ✅ Input validation and error handling
- ✅ In-memory database (no external setup required)
- ✅ CORS support for cross-origin requests

## Quick Start

```bash
cd TaskManagerAPI
dotnet build
dotnet run
```

The API will be available at `http://localhost:5000`
